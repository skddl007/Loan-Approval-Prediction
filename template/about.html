About My Website

BorrowBuddy is a web application designed to predict loan amounts for individuals based on various factors. Our system utilizes machine learning models, specifically linear regression and random forest, to analyze data and provide accurate loan predictions.

Whether you're an individual looking for an estimate on a personal loan or a financial institution analyzing loan data from a CSV file, our platform offers reliable predictions to assist you in making informed decisions.

Features:

  • Utilizes linear regression and random forest machine learning models
  • Provides accurate loan predictions
  • Easy-to-use interface for inputting data
  • Supports analysis of data from CSV files

At BorrowBuddy, our goal is to simplify the loan prediction process and offer a valuable tool for both individuals and professionals in the financial sector.
